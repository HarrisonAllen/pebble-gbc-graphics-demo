Notes:
  • Sprite data that is fetched from non-ROM memory is simulated via a snapshot of memory ($8000-$FFFF) while checking a Pokémon's status screen.
  • The snapshot was taken from the US release of Pokemon Red, so sprites from other games may not be accurate from this region of memory.
  • The memory region where the decompressed data is temporarily stored is not simulated. All reads return zero, even if the buffer has data in it.
